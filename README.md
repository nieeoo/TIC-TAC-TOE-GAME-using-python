# TIC-TAC-TOE-GAME-using-python
This is a console-based Tic-Tac-Toe game developed in Python for two players. The game uses a 3×3 board where players take turns placing X and O. It validates user input, updates the board, checks for winning combinations, and declares a winner or tie. The project demonstrates Python basics such as functions, loops, lists, and conditionals.
